# tmux-nvim
This is a script to setup T-MUX and NeoVIM


## Installation
```sh
curl -L https://raw.githubusercontent.com/mrasif/tmux-nvim/main/install.sh | bash
```

## Uninstall

```sh
curl -L https://raw.githubusercontent.com/mrasif/tmux-nvim/main/uninstall.sh | bash
```
